To run the Translator Program:

(1) You need to download the latest Stanford NLP POS tagger package.
    a. Visit http://nlp.stanford.edu/software/tagger.shtml
    b. Download POS tagger version 3.3.1 2014-01-04	(full) to code/ folder
    c. Unzip the file, to code/ folder
    d. rename the package directory as "stanford-postagger-full"

(2) Make sure you are in code/ directory

(3) Execute the following:
bash run-translator.sh
